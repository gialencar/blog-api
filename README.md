# Blog API

# Contexto
Este projeto trata-se de uma API de um CRUD para posts de blog. 

## Técnologias usadas

Back-end:
> Desenvolvido usando: NodeJS, ExpressJS, MySQL, Sequelize


## Instalando Dependências

> Backend
```bash
cd blog-api/
npm install
``` 

## Executando aplicação

* Para rodar o app:

  ```
  npm start
  ```

## Executando Testes

* Para rodar todos os testes:

  ```
    npm test
  ```
